Core Research Questions

  • Does a voice AI built to detect emotional expression register autistic vocal expression with the same accuracy it shows for neurotypical expression — or does it inherit the same bias documented in facial-recognition systems trained mostly on neurotypical data?
  • Where the machine is wrong, is it wrong in a consistent direction — always mistaking one feeling for the same other feeling — or randomly?
  • Could a machine, in some specific instance, register something true about an autistic speaker's feeling that a neurotypical listener misses?

Theoretical Framework

[Note collaborator's name and discipline here.] This project keeps the same governing rule that shaped the punctuation work: two independent investigations, run side by side, neither one explaining or standing in for the other.

The embodied side of this work has to come from autistic collaborators directly, not from an artist's reflection on their behalf — that substitution would be a different, weaker claim than the punctuation piece made, where any person's relationship to a comma was a fair stand-in for "a person's." The machine side stays governed by the same rule as before: EVI is never asked to discuss or interpret emotion or autism. It is only ever observed.
Asking a model to self-report on emotion produces performed language about emotion, not a machine signal — the same finding that shaped the punctuation piece's method. Here that means working from EVI's actual classification and confidence output against real speech, never from EVI's own description of what it thinks it heard.

Machine Behaviour

EVI's classification and confidence scores against real speech, extracted as data — not self-report, the same principle as the punctuation piece's machine track.

The Channel

Not the keyboard this time — the voice. Prosody: tune, rhythm, timbre. The place tone of voice passes between a person and a machine that's listening for feeling rather than words.

The Gap

Where the true emotion and EVI's reading agree, something about the mark's legibility is revealed. Where they diverge — and in which direction — something about the machine's training bias becomes visible instead.

Methodology

Investigative Process

1Fix the Ground-Truth Taxonomy

Decided with the collaborator, not chosen artistically. This has to be settled before anything else — it is what makes the "true" label valid.

2Source Consented Expression Samples

Real autistic speakers, anonymised, gathered through the collaborator's ethics pathway.

3Run Samples Through EVI — Behaviour, Not Self-Report

EVI 3, potentially configured on Claude as the underlying model. Capture the actual classification and confidence score per sample.

4One Vessel Per Baseline Emotion

Each vessel carries a base pattern for its emotion. A translucent tracing-paper hand-shape, in the confused emotion's gestural form, overlaps onto the vessel wherever EVI mistakes one for the other — layered more heavily where the confusion is frequent. Confusion is recorded as asymmetric: each direction shown independently.

5Sonify the Comparison, Per Emotion

Two simultaneous lines per vessel's sound piece — true emotion and EVI's detected emotion across the test timeline. Pitch identifies the emotion category. Confidence maps to volume and clarity, not pitch, so a confident misread sounds clean and certain rather than hesitant. Density of divergence becomes the piece's overall texture.

6Pair Object and Sound by Proximity

Same mechanism as the punctuation piece. Approaching a vessel triggers that emotion's sound. The visible bleed and the audible clash should corroborate one finding, not duplicate it.
